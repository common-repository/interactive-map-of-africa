== Description ==
Download this interactive map of Africa WordPress plugin and customize the map colors, links, hover description for each of the 5 african regions through an easy to use admin panel without any coding knowledge, then insert the map shortcode in any page/post to display the map.

== Installation ==

1. Install the plugin .zip file through your WordPress 'Plugins' screen directly, or Upload the plugin files to the `/wp-content/plugins/interactive-map-of-africa` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Now, you will see a new menu button in the left side bar of your WordPress called 'Africa Reg. Map', click it and customize your map then save the changes.
4. Use this shortcode `[afr_map]` to display the map in any page or post.

== Frequently Asked Questions ==

= Can I add an image in the hover popup? =

Yes, you can write any text and add images.

= Does this map requires Flash to work? =

No, it's SVG based and works perfectly on desktop along with mobile devices 'smartphone'.

= Is this map responsive? =

Yes.

= Can I split region into countries? =

No, but fortunately we already have a [Premium Interactive Map of Africa](https://www.wpmapplugins.com/interactive-map-of-africa-wordpress-plugin.html) with clickable African countries.

= Can I set a region such as Northern Africa as inactive? =

Yes, simply un-check the (active) box beside that region to make it inactive.

= Can I use the shortcode in different pages to display the same map? =

Yes.

= Can I create a different map and use it in another page? =

No, but you can buy the **[Multiple Instances Regional Map of Africa Package](http://wpafr.html5maps.pay.clickbank.net/)** to get 3 instances of the plugin and install each one to get a different styled map.

= Can I link a region to any webpage? = 

Yes, also you can choose to open it in the same window or in a new window.

Yes, we can customize the map for what you need as a freelance customization service, just [contact](https://www.wpmapplugins.com/interactive-map-of-africa-wordpress-plugin.html#contact) us.
